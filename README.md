# Desafio DIO - Dominando a Linguagem de Programação Swift

Este é um desafio proposto pela [DIO](https://www.dio.me/) na "[Formação iOS Developer](https://www.dio.me/curso-ios-developer)", no módulo "Dominando a Linguagem de Programação Swift".

O desafio consiste em:

- Criar dois projetos no playground usando o Xcode.
- Para o primeiro projeto aplicar o Paradigma de Programação Orientado a Objetos.
- No segundo projeto, aplicar o Paradigma de Programação Orientada a Protocolos.
